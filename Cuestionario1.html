Preguntas

[1]
  1. ¿Qué es una base de datos?
    • Una base de datos es un conjunto de archivos que tienen un cierto tipo de relación entre ellos.

  2. ¿En qué casos es conveniente usar bases de datos?
    • Depende de la metodología en la que se soluciona el proyecto. En el caso de que se tenga que tratar con grandes cantidades de información, es una buena práctica implementar una base de datos para que de ésta manera el sistema pueda gestionarlo de una mejor manera. En resumen, se requiere usar una base de datos cuando ocupas relacionar la información, o cuando existen grandes bloques de información.

  3. ¿Qué es un sistema de gestión de base de datos - DBMS?
    • Es una técnica en la cual se trabaja con grandes bloques de información para relacionarlos entre si. Sus siglas en ingles quieren decir "Database Management System". Éste método consiste en tomar una conjunto de datos relacionados (base de datos) y se utilizan ciertos programas para acceder y colectar esos datos.

  4. Enlista y explica las funciones/responsabilidades que tienen los DBMS.
    • Ofrecer seguridad a la base de datos, pues la misma tiene un valor y, de ser necesario, éstos deben tener un acceso restringido para ciertas personas.
    • Los bloques de datos deben de tener cierto tipo de limitaciones, naturalmente basándose en las normas de la organización, por ejemplo, que a un empleado se le debe pagar 50 dólares la hora.
    • Un DBMS siempre debe estar preparado para la mayor cantidad de problemas que deben surgir, por ejemplo, si por alguna razón se pierde la información, ésta debe de tener una copia de seguridad y la capacidad de acceder a la misma en caso de emergencia.
    • Es muy común que los usuarios del DBMS tengan que acceder y modificar la base de datos al mismo tiempo, es por eso que el DBMS debe de estar preparado para evitar que sucedan discrepancias a la hora de que sucedan estas concurrencias.

  5. Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.
    • Yo siento que hubiera sido una buena idea implementar una base de datos en un par de proyectos que he tenido a lo largo de mi carrera. Por ejemplo, cuando cursaba la carrera de diseño interactivo diseñé una página web orientada al aprendizaje del lenguaje de señas, opino que hubiera tenido un mejor alcance si se le implementa una base de datos para que el usuario tenga su propia sesión y pueda ver el progreso de su aprendizaje.
    • Otro proyecto en el que hubiera implementado una pequeña base de datos es mi proyecto de videojuegos. El proyecto hubiera tenido un alcance mayor si al mismo se le agrega un scoreboard donde los usuarios puedan ver al mejor jugador en una tabla de posiciones.
