  1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?
    • Primero, Java es un lenguaje que requiere de un compilador para poder ser ejecutado; mientras que JavaScript es un lenguaje que no se compila, sino que es interpretado por un navegador.
      A diferencia de Java, JavaScript tiene características que le permiten trabajar con una misma variable de distintas maneras, es decir, puedes declarar una variable y utilizarla como un entero en un principio y al final puedes utilizarla como un caracter.

  3. ¿Qué métodos tienen los arreglos? (Menciona al menos 5*) [2]

  4. ¿Cómo se declara una variable con alcance local dentro de una función?
    • Colocando la declaración "var" dentro de la función. Aunque también se puede utilizar "let".

  5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?
    • Perdería el propósito de lo que es una función, lo cual es encapsular el procedimiento para que de esa manera exista una mejor cohesión.

  6. ¿Qué método de String se puede utilizar para buscar patrones con expresiones regulares?¿Para qué podrías utilizar esto en una aplicación web?
    • Utilizaría el método match() el cual te ayuda a buscar strings dentro de otros strings y los añade a un arreglo. Se podría utilizar para un buscador en tu página web para buscar frases o palabras exactas. [3]
