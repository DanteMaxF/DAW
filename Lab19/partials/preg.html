Preguntas

  1. ¿Qué importancia tiene AJAX en el desarrollo de RIA's (Rich Internet Applications)?
    • AJAX es una técnica de Javascript que le permite al usuario tener una mejor experiencia cuando se encuentra en la página web. Pues permite tener una interacción de "tiempo real", en donde el usuario no tiene que refrescar la página, además de que es otra manera de acceder a la base de datos de una manera dinámica.

  2. ¿Qué implicaciones de seguridad tiene AJAX? ¿Dónde se deben hacer las validaciones de seguridad, del lado del cliente o del lado del servidor?
    • Como los datos son enviados en texto plano por medio de un GET. Una manera de evitar los ataques es mediante la validación de datos desde el servidor.

  3. ¿Qué es JSON?
    • Es un texto por el que se manda la información de un objeto, tales como sus atributos.
