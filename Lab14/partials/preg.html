Preguntas

  1. ¿Qué es ODBC y para qué es útil? [1]
    • ODBC es la abreviación de Open DataBase Connectivity, la cual es un estándar desarrollado para que las aplicaciones que accedan a las bases de datos puedan trabajar con cualquier motor con la única necesidad de cambiar sólo un elemento.

  2. ¿Qué es SQL Injection?
    • Es un modo de ataque a los sitios web en donde el hacker ejecuta scripts de SQL a travéz de las formas que se encuentran en el sitio.

  3. ¿Qué técnicas puedes utilizar para evitar ataques de SQL Injection?
    • Una manera para evitar éste tipo de ataques, es creando verificaciones en php para que el sistema no ejecute las entradas como un script de SQL.
